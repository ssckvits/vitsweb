Web Development

  • Build a website centered around a fictional networking and telecommunication company (ex: Dialog).
  • Categories:
    • Grade 6 - 9
    • Grade 10 - 13
    • Each school can submit up to 2 entries per category, totaling 4 submissions.

Instructions

  • Pages Required(Additional Pages are allowed):
    • Home: Introduction to the company, including a brief overview of services.
    • Plans: Detailed descriptions of various service plans and pricing.
    • Customer Support: Contact form, FAQs, or support options for customers.
  • This is an individual event. Each school is allowed two submissions per age category.
  • Your website will be judged based on the following criteria:
    • Design & Creativity
    • Functionality & User Experience
    • Technical Skills

Rules

  • All code must be hand-coded using HTML, CSS, and JavaScript.
  • The use of content management systems (CMS) or pre-designed templates is strictly prohibited. All code must be original.
  • The website must be responsive and functional on both desktop and mobile devices.
  • The website should be compatible with the latest versions of major browsers (e.g., Chrome, Firefox, Safari, Edge).
